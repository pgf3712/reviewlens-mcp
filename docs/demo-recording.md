# Demo recording and README media

The repository already includes a lightweight animated walkthrough at
`docs/images/reviewlens-demo.gif` and four full-resolution screenshots in the
same directory. The original MP4 is source material and should not be committed.

## Recording a replacement

1. Use a 1440×900 browser window at 100% zoom. Hide bookmarks, notifications, and personal tabs.
2. Start in English at the top. Pause 3 seconds on the headline and trace card.
3. Scroll to Interactive demo in 2 seconds.
4. Click Run review once. Pause 7–10 seconds on risk, evidence, tests, and the injection signal.
5. Scroll through Tool explorer and Architecture for 8–10 seconds.
6. End on Security by design for 5 seconds.
7. Optional: switch to ES for 3 seconds.

Record at 30 fps with no audio and export MP4 first. Keep the repository GIF at
960 px wide and under 8 MB. Replace `docs/images/reviewlens-demo.gif` only after
checking that text remains readable on GitHub.
